# Intelligent Resume Analyzer

## Overview
A Python-based system that analyzes resumes against job descriptions and calculates a match score using rule-based scoring and skill extraction.

## Features
- Resume parsing
- Skill extraction
- Job description matching
- Match score calculation
- Clean CLI interface

## Tech Stack
- Python 3
- Regex / NLP
- JSON processing

## Setup Guide

1. Clone repo
```sh
git clone https://github.com/Vatsal08Xr/Intelligent-Resume-Analyzer_HiDevs
```

2. Install dependencies
```sh
pip install -r requirements.txt
```

3. Run
```sh
python main.py resume.txt
```
